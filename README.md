### Installation de Brew si tu n'as pas 
```bash
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew update
export PATH="/usr/local/bin:$PATH" >> ~/.zshrc
source ~/.zshrc
```
### Installation de NodeJS et lancement
```
brew install node
cd backend
npm install -s
npm run start
```
